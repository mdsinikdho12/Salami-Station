## 📖 প্রজেক্ট সম্পর্কে

**সালামি স্টেশন** হলো ঈদুল ফিতর উপলক্ষে তৈরি একটি মজাদার ওয়েব অ্যাপ্লিকেশন। ঈদের দিন বন্ধু, ভাই, দুলাভাই বা ধনী আত্মীয়দের কাছ থেকে সালামি চাওয়াটা এখন আরও সহজ ও মজাদার! কাস্টম সালামি রিকোয়েস্ট কার্ড তৈরি করুন, লিংক শেয়ার করুন এবং সালামি পান।

> *"বন্ধু, ভাই, দুলাভাই ও ১৪ গোষ্ঠীর কাছে সালামি চান — নির্ভয়ে। সালামি পাওয়ার দায়িত্ব আমাদের।"* 🥱

- 🔗 **শেয়ারযোগ্য লিংক** — একটি ইউনিক লিংক তৈরি হয় যা WhatsApp, Messenger ও Facebook-এ শেয়ার করা যায়
- 💸 **বিকাশ ইন্টিগ্রেশন** — বিকাশ নম্বর সরাসরি কার্ডে দেখায়, এক ক্লিকে কপি করা যায়
- 🎁 **Founder কে সালামি** — প্রজেক্ট ক্রিয়েটরকেও সালামি দেওয়ার অপশন
- 📱 **মোবাইল ফার্স্ট ডিজাইন** — যেকোনো ডিভাইসে পারফেক্ট দেখায়
- 🌙 **ঈদ থিম** — ঈদের আবহ বজায় রেখে সুন্দর UI/UX ডিজাইন
- ⚡ **দ্রুত ও হালকা** — Vercel-এ হোস্ট করা, লোড টাইম অত্যন্ত কম

> **Note:** মেইন রিপোজিটরি প্রাইভেট। শুধুমাত্র কনট্রিবিউটরদের জন্য অ্যাক্সেস পাওয়া যাবে।

```bash
# রিপো ক্লোন করুন (অ্যাক্সেস থাকলে)
```
